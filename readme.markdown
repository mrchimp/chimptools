Chimptools
==========

A collection of cross-platform command line tools.
--------------------------------------------------

save.py
=======

    save [--help] [--public || --private] *word* *content* *[content ...]*


show.py
=======

    show [--help] [--words] [--public | --private] [--mine] [--distinct] [word] [filter]

umm.py
======

    umm [--help] [--public | --private] [--mine] search_term
